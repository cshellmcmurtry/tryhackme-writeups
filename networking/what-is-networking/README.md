## Objective

This room introduced the fundamentals of computer networking, including how devices communicate with one another, the purpose of networks, and the components that make communication across a network possible.

## Key Concepts

### What is a Network?

A network is a collection of devices connected together so they can communicate and share resources such as files, printers, applications, and internet access.

### Types of Networks

Common network types include:

- LAN (Local Area Network)
- WAN (Wide Area Network)
- MAN (Metropolitan Area Network)
- PAN (Personal Area Network)

Each network type is designed to serve different geographic areas and communication needs.

### Network Devices

Common networking devices include:

- Routers
- Switches
- Firewalls
- Access Points
- Modems

Each device plays a different role in directing, securing, or transmitting network traffic.

### IP Addresses

An IP address is a unique identifier assigned to a device on a network. It allows devices to locate and communicate with one another.

### Network Communication

Devices communicate by sending and receiving data in the form of packets. Networking protocols define how these packets are transmitted and received across a network.
