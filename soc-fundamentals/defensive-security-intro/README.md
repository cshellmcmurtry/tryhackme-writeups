## Objective

This room introduced the fundamentals of defensive cybersecurity and explained how security professionals work to detect, analyze, and respond to cyber threats. It also provided an overview of the tools and practices used to protect systems and networks from attacks.

## Key Concepts

### What is Defensive Security?

Defensive security focuses on protecting an organization's systems, networks, and data from cyber threats. It involves identifying vulnerabilities, monitoring for suspicious activity, and responding to security incidents before they can cause significant damage.

### Threat Detection

Threat detection involves continuously monitoring systems and networks for unusual or malicious activity. Security teams use a combination of security tools, alerts, and threat intelligence to identify potential attacks.

### Incident Response

Incident response is the process of identifying, containing, investigating, and recovering from a security incident. Having a structured response process helps organizations minimize damage and restore normal operations quickly.

### Security Monitoring

Security monitoring is the continuous observation of systems, applications, and network traffic to identify indicators of compromise and other suspicious behavior.
