## Objective

This room introduced the responsibilities of a Tier 1 Security Operations Center (SOC) analyst and explained how security alerts are investigated, triaged, and escalated.

---

## Key Concepts

### What is a SOC?

A Security Operations Center (SOC) is a team responsible for monitoring an organization's systems for suspicious activity, investigating security events, and responding to incidents.

---

### What is a SIEM?

A Security Information and Event Management (SIEM) platform collects logs from multiple systems, correlates events, and generates alerts when suspicious activity is detected.

Examples include:

- Splunk
- Microsoft Sentinel
- IBM QRadar
- Elastic

---

### What is Alert Triage?

Alert triage is the process of determining:

- Is the alert legitimate?
- How severe is it?
- Does it require escalation?
- What systems or users are affected?
