## Objective

This room introduced the role of a Security Operations Center (SOC) within a Blue Team environment. It explained how SOC analysts monitor systems, investigate alerts, respond to security incidents, and work together to protect an organization's assets.

## Key Concepts

### What is Blue Team?

A Blue Team is responsible for defending an organization's systems, networks, and data from cyber threats. Their primary focus is on detection, prevention, monitoring, and incident response.

### What is a Security Operations Center (SOC)?

A Security Operations Center (SOC) is a centralized team that continuously monitors an organization's environment for suspicious activity, investigates alerts, and responds to security incidents.

### SOC Analyst Responsibilities

Common responsibilities of a SOC analyst include:

- Monitoring security alerts
- Investigating suspicious activity
- Analyzing security logs
- Triaging alerts
- Escalating incidents when necessary
- Documenting findings
- Supporting incident response efforts

### Blue Team Tools

Blue Teams commonly use tools such as:

- SIEM platforms
- Endpoint Detection and Response (EDR)
- Firewalls
- Intrusion Detection Systems (IDS)
- Intrusion Prevention Systems (IPS)
- Threat Intelligence platforms
